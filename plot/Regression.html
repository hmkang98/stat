이 프로그램은 동적화면에서 산점도와 상관계수의 대응관계비교할 수 있다. 다음은 이 프로그램 사용법이다.

  • 이 프로그램에서 자료는 텍스트 파일을 열거나 화면에서 사용자가 마우스를 클릭하여 사용할 수 있다.
  • 파일에서 부른 자료는 최초로 생성된 창 크기의 95%에 산점이 모두 찍히도록 하였다.
  • 자료의 삭제는 마우스를 드래그-앤-드롭(Drag-and-Drop)하여 그 영역에 포함된 산점을 제거할 수 있다.
  • 자료의 추가, 삭제하는 경우 변경된 자료에 대한 결과를 즉시 확인할 수 있다.
  • 산점이 찍힌 영역에 마우스가 놓이면 그 산점의 좌표, 잔차, 스트던트화 잔차(studentized residual)와 유의확률(p-value), 쿡 통계량(Cook's Distance)과 유의확률을 화면에 보이므로 이상치(outlier)와 영향력 관측치(influential observation)를 확인할 수 있다.
  • 마우스가 그 영역을 벋어나면 그 산점에 대한 정보와 통계량이 사라진다.
  • 또한 좌측 하단에 피어슨(Peason) 표본 상관계수와 스피어만(Spearman) 순위 상관계수가 출력된다. 또한 분산분석표, 단순회귀직선, 신뢰대(Confidence Band), 예측구간의 신뢰구간(Prediction Band) 등도 제공된다.
이 프로그램을 사용시 요구사항은 JRE가 설치되어 있어야 한다. 만일 JRE가 설치되지 않은 컴퓨터는 자동으로 JRE를 설치할 수 있는 장소로 연결해 준다. 펜티엄 4 컴퓨터를 기준으로 이 프로그램 설치의 소요시간은 약 10분 미만이다.

  1. Data : 자료 입력의 시작 방법과 인쇄, 저장의 서브메뉴가 존재한다.
    • Mouse : 마우스 클릭으로 자료를 입력하기 시작한다. 자료 편집창에서 자료의 수정, 추가, 삭제가 가능하다.
      1. Input : 마우스 클릭으로 자료를 생성하는 메뉴이다.
      2. Save Points : 마우스 클릭으로 생성된 자료를 저장하는 메뉴이다.
    • File : 자료 파일에서 자료를 읽거나 저장하는 메뉴이다. 파일을 읽은 후 마우스를 이용하여 자료의 추가, 삭제, 수정이 가능하다.
      1. Open : 읽을 자료 파일을 지정하는 메뉴이다.
      2. Save : 자료를 파일로 저장하는 메뉴이다.
    • Print : 동적그래픽 화면을 프린터로 출력하는 메뉴이다.
    • Exit : 프로그램을 종료한다.
  2. Draw : 회귀직선, 신뢰대, 예측구간 관련 메뉴이다.
    • Line : 회귀직선을 그리는 메뉴이다.
    • Confidence Band : 신뢰대 $\hat{E}(Y|X=x)$를 그리는 메뉴이다.
    • Prediction Band : 예측구간 $(\hat{Y}|X=x)$를 그리는 메뉴이다.
  3. Statistic : 관련 통계량을 출력하는 메뉴이다.
    • Anova : 분산분석표, 계수의 추정 및 검정 값 등을 출력하는 메뉴이다.
