A browser with JavaScript enabled is required for this page to operate properly. 정규분포에서 분위수(Quantile)및 유의확률(P-value)구하기
  1. 분위수를 구하는데 사용한 알고리즘
    • Hasting(1955)가 제시한 알고리즘 사용.
    • 정확도 : 소수점 이하 7자리.
  2. 사용방법
    • "정규분포 시작" 버튼을 누른다.
    • 평균 : 모든 실수 가능
    • 분산 : 0이상의 실수 입력
    • 신뢰도 : 0에서 100사이의 실수 입력
    • 분위수 : 정규분포로 표준화한 값이 -10~10사이의 값 입력
    • "다시그리기" 버튼을 누른다.
  3. 결론
    • 두 모수가 변함에 따른 그래프의 동적 변화를 알 수 있다.
    • 신뢰도가 변할때 정규분포의 임계값을 알 수 있다.
    • 양쪽 빨강색의 크기는 유의확률이다.
