통계응용프로그램 실행 환경

이 사이트에 구현한 통계프로그램은 자바 언어로 만들었다. 따라서 응용프로그램을 실행하려면 로컬 컴퓨터에 JRE가 설치되어 있어야 한다. JRE 설정 후 어플리케이션 보안을 허용해야 프로그램이 실행된다.

자바프로그램 설치와 어플리케이션 보안 허용 설정

  1. 자바 홈페이지에서 자바 프로그램(Java Runtime Envirnment)을 설치한다. 아래 프로그램이 JRE이다.

  2. 자바 보안설정

    1. 제어판에서 자바 제어판 실행

    2. 보안 탭에서 사이트 목록 편집 클릭

    3. 예외사항 사이트 목록에 https://hmkang98.github.io 추가하고 확인버튼을 클릭
